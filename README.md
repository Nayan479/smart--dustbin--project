## 📌 What is smart--dustbin--project?

The smart--dustbin--project is an IoT-based system designed to make waste disposal simple and clean. It uses sensors to detect when a hand or trash is near. The dustbin lid opens automatically to avoid touching it. This helps keep places neat and reduces germs.

The system works well in homes, offices, parks, and other public areas. It limits the spread of dirt and bad smells by removing the need to touch the dustbin.

## 🧩 How the Smart Dustbin Works

The smart dustbin uses an ultrasonic sensor. It detects objects like your hand or waste bags near the bin.

The sensor sends signals to the Arduino Uno microcontroller. The Arduino processes the data and commands a servo motor to open the lid automatically. This process is fast and silent.

This system removes the need to touch the bin, keeping it clean and safe from germs.
