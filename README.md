# Smart Dustbin – IoT Based Automatic Waste Management System

## 📌 Project Overview
The Smart Dustbin is an IoT-based automatic waste management system designed to improve hygiene and cleanliness in public and private places. The system uses sensors to detect human presence and automatically opens the dustbin lid, reducing manual contact and promoting sanitation.

## 🎯 Problem Statement
Traditional waste collection methods are unhygienic and inefficient. Overflowing dustbins cause pollution, bad odor, and health issues. Manual handling of waste increases the risk of spreading diseases.

## ✅ Objectives
- Automatic opening of the dustbin lid
- Touchless and hygienic waste disposal
- Low-cost and easy-to-maintain system
- Suitable for smart city applications

## 🧠 Working Principle
An ultrasonic sensor detects the presence of an object (hand or waste) near the dustbin.  
The Arduino Uno processes the sensor data and sends a control signal to the servo motor.  
The servo motor opens the dustbin lid automatically and closes it after a short delay.

## 🔧 Components Used
- Arduino Uno  
- Ultrasonic Sensor (HC-SR04)  
- Servo Motor  
- Breadboard  
- Jumper Wires  
- Power Supply  

## 🔌 Circuit Diagram
The circuit connects the ultrasonic sensor and servo motor to the Arduino Uno.  
The sensor handles object detection, while the servo motor controls lid movement.
